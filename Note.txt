AngularJS notes:
_expression = {{something}} format
_module= container for an application
var contentApp = angular.module('contentApp', []);

_directive: extending html attribute with "ng-"
+ng-app -> init the application to use Angular
+ng-model -> bind the value of HTML control (such as input) to the app data (firstname)
+ng-repeat -> use with loop to duplicate the HTML element

_model / ng-model (directive) : bind value of input field to var of Angular (bi-direction)
+can use to validate format of a var 

_binding / ng-bind (directive): bind value to something (unlike model, doesn't have to be a field)

_controller / ng-controller 
<div ng-app="myApp" ng-controller="myCtrl">
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
	something
}

_$location: used to save window.href location
